歌もトークも、これ一本で。声を整えて、音程を直して、遊べる。配信でもそのまま使える遅延ゼロ設計のボーカルチャンネルストリップです。 One plugin for singing and talking. Clean up the voice, fix the pitch, then play with it — a vocal channel strip that adds no latency, so you can leave it on while you stream.

できることWhat it does

声にまつわる作業を、一つの画面で。Everything the voice needs, on one screen.

ノイズ取りから音程補正、声変えまで。プラグインを何枚も重ねなくても、配信や録音でよく使う処理が順番に並んでいます。 From noise removal to pitch correction and voice changing — the processing you reach for while streaming or recording, in the order you use it.

オートチューンAuto-tuneNEW 1.9.0

歌のピッチを検出して、選んだキー／スケールへ自動で補正。フォルマントを保つので声の性別感は変わりません。「速さ」でケロケロ〜自然を調整できます。 Detects your pitch and snaps it to the key and scale you pick. Formants stay put, so the voice keeps its character. The Speed control takes you from hard robotic steps to a natural glide.

ノイズ除去（LEARN）De-noise (LEARN)

黙ったまま LEARN を押すと約1.5秒で部屋のノイズを学習。4帯域の適応型で、エアコンや PC のファン音を削ります。遅延は増えません。 Stay quiet, press LEARN, and it profiles your room in about 1.5 seconds. Four adaptive bands then pull down fan and air-conditioner noise, with no added latency.

声質・マイク・シーンVoice, mic and scene presets

声質10種 × マイク10種 × シーン3種（弾き語り／トーク配信／バンド）を掛け合わせるだけで土台ができます。ワンプレス自動設定も。 Ten voice types × ten microphones × three scenes (solo, talk stream, band) combine into a starting point. There is also a one-press auto setup that listens and decides for you.

2段コンプ＋ディエッサーTwo-stage compression + de-esser

ピーク圧縮と平準化を分けて、音量差の激しい歌でも聴きやすく。耳に刺さる「サ行」だけを狙って抑えます。 Peak control and levelling run as separate stages, so loud phrases sit with quiet ones. The de-esser targets sibilance only.

声変え・5人ユニゾンVoice change and 5-voice unison

ピッチとフォルマントを別々に動かせるので、性別感だけ変えることもできます。5人ユニゾンとハモリでコーラスも作れます。 Pitch and formant move independently, so you can shift the character without changing the note. Add the five-voice unison and harmony for an instant backing group.

空間・キャラ声Space and character FX

リバーブ（スプリング／シマー含む）、テンポ同期ディレイ、ダブラー、メガホン、ロボ声。歌でも雑談でも遊べます。 Reverb (including spring and shimmer), tempo-synced delay, doubler, megaphone and robot voice — for songs and for goofing around on stream.

画面The screen

ツマミはぜんぶジュース。Every knob is a drink.

中身は本格的でも、見た目はやさしく。文字サイズと画面モード（ゆるふわ／自然／ブランド／オーロラ／軽量／色覚）を切り替えられます。 Serious processing, friendly surface. Text size and screen mode (soft, seasons, brand, aurora, light, colour-blind safe) are all switchable.

インストールInstall

3ステップで使えますThree steps

インストーラーが VST3 と単体起動版を入れます。手動でコピーしたい場合は、VocalGzzio.vst3 フォルダを C:\Program Files\Common Files\VST3\ に置くだけでも動きます。 The installer places the VST3 plugin and the standalone app. If you prefer to do it by hand, copy the VocalGzzio.vst3 folder into C:\Program Files\Common Files\VST3\.

  1. セットアップを実行するRun the setup

    Releases から VocalGzzio_Setup_v1.9.0.exe をダウンロードして開きます。 Download VocalGzzio_Setup_v1.9.0.exe from Releases and open it.

  2. SmartScreen の警告を通過するGet past SmartScreen

    青い画面が出たら「詳細情報」→「実行」。コード署名をしていないためで、異常ではありません。 If the blue screen appears, choose “More info” then “Run anyway”. The installer is unsigned; that is the only reason for the warning.

  3. DAW を再起動して探すRestart your DAW and find it

    FX 追加から「VocalGzzio」を検索。出てこないときはプラグインの再スキャンをしてください。 Search for “VocalGzzio” in the plugin browser. If it does not appear, run a plugin re-scan.

すでに入っているときは、4つから選べますAlready installed? Pick one of four

同じバージョンやファイルが見つかると、インストーラーが最初に選択画面を出します。 When the installer finds an existing copy, it opens with this choice.

更新Update 今の構成のまま、新しいバージョンに入れ替えます。 Replace the installed files with the newer version, keeping the same components.
修正Repair 壊れた・消えたファイルを同じバージョンで入れ直します。 Reinstall the same version to restore missing or damaged files.
追加・変更Add or change VST3 と単体起動版のうち、入れるものを選び直します。 Choose again which parts to install: the VST3 plugin, the standalone app, or both.
削除Remove アンインストールします（設定ファイルは残ります）。 Uninstall it. Your saved settings stay on the machine.

ソースからビルドBuild from source

3行でビルドできますThree commands

JUCE 8.0.12 は CMake が自動で取ってくるので、事前に用意する必要はありません。初回の設定だけ数分かかります。 CMake fetches JUCE 8.0.12 for you, so there is nothing to install by hand. Only the first configure step takes a few minutes.

必要なもの（Windows）You need (Windows)

  • Visual Studio 2022 （C++ デスクトップ開発）(Desktop development with C++)
  • CMake 3.22+
  • Git
  • インストーラーを作るなら Inno Setup 6Inno Setup 6, if you want the installer

macOS / LinuxmacOS / Linux

  • macOS: Xcode + CMake
  • Linux: build-essential, libasound2-dev, libx11 系, libfreetype6-dev, libfontconfig1-dev, libgl1-mesa-dev Linux: build-essential, libasound2-dev, the libx11 family, libfreetype6-dev, libfontconfig1-dev, libgl1-mesa-dev
  • GitHub Actions で Windows と macOS を自動ビルドしていますGitHub Actions builds Windows and macOS on every push
# Windows
git clone https://github.com/YOUR-ACCOUNT/VocalGzzio.git
cd VocalGzzio
cmake -B build -A x64
cmake --build build --config Release
# macOS / Linux
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --parallel 4

出力先：build/VocalGzzio_artefacts/Release/VST3/VocalGzzio.vst3（単体起動版は同じ階層の Standalone/）。インストーラーは Installer.iss を Inno Setup で開いて Compile すると installer_output/ に出ます。 Output lands in build/VocalGzzio_artefacts/Release/VST3/VocalGzzio.vst3, with the standalone app beside it. For the installer, open Installer.iss in Inno Setup and press Compile; the result appears in installer_output/.

更新履歴Changelog

どこがどう変わってきたかWhat changed, and when

新しい順。各バージョンの詳しい変更点は、リポジトリ直下の v1.x.x_◯◯_変更点.md に置いてあります。 Newest first. The full notes for each release live in the repository root as v1.x.x_…_変更点.md.

  1. v1.9.0

    オートチューンと、SONAR で認識されない問題の修正Auto-tune, and the SONAR detection fix

    • オートチューンを追加：YIN でピッチ検出 → キー／スケール（9種）へスナップ → 声変えと同じシフターで補正。フォルマントは保持。 Added auto-tune: YIN pitch detection, snapping to a key and one of nine scales, applied through the same shifter as the voice changer. Formants are preserved.
    • 「キー検出」の結果がオートチューンのキーとスケールに自動で入るようになりました。 The key-detect button now fills in auto-tune's key and scale for you.
    • 実測：補正誤差 平均 0.24 セント／THD 0.11%／CPU 追加分 +2.4%（声1本）。 Measured: 0.24 cents average correction error, 0.11% THD, and 2.4% extra CPU per voice.
    • SONAR 2025.12 でプラグインが出てこない問題を修正（VST3 マニフェストの生成を止め、ホスト側の通常の読み込み経路に戻しました）。 Fixed the plugin not appearing in SONAR 2025.12 by switching off the VST3 manifest, so hosts fall back to reading the binary directly.
  2. v1.8.5

    夏モードのパネルを透過See-through panels in summer

    • 自然モードの夏で、パネル越しに海と空が見えるように半透明化（ツマミの見やすさは維持）。 Panels in the seasons mode turn translucent in summer so the sea and sky show through, while the knobs stay readable.

    声変えの音質を作り直しRebuilt the pitch and formant engine

    • ピッチシフトを正統な位相ボコーダ方式へ変更、FFT を 512→1024 に拡大。ざらつきが大きく減りました。 Moved pitch shifting to a proper phase-vocoder method and doubled the FFT size from 512 to 1024, which cleared up most of the roughness.
    • 夏の背景を「海と空」と分かる絵に描き直し。 Redrew the summer background so it reads clearly as sea and sky.
  4. v1.8.3

    • 大きな飾りを全廃して背景＋小さな粒子に。「オーロラ」モードを追加し、ヘッダーの重なりを解消。 Dropped the large decorative motifs in favour of a background with small particles, added the aurora mode, and fixed overlapping header elements.

  7. v1.8.0

  8. v1.7.0

  9. v1.6.0

    • スプリングとシマーを追加し、リバーブ種別の選択を常に触れる位置へ移動。出力に合わせて回るギアのメーターを追加。 Added spring and shimmer reverbs, moved the reverb type selector somewhere you can always reach it, and added the gear meter that spins with your output.

FAQ

よくある質問Questions

DAW にプラグインが出てきませんMy DAW does not list the plugin

プラグインの再スキャンを実行してください。SONAR 2025.12 で出てこなかった問題は v1.9.0 で修正済みです。古い版をお使いなら更新してください。 Run a plugin re-scan first. If you are on SONAR 2025.12, update to v1.9.0 — that release fixes the case where the plugin was silently skipped.

オートチューンはどう設定すればいいですかHow should I set up auto-tune?

キーとスケールを曲に合わせ、補正 80%・速さ 30% から始めてください。ケロケロにしたいときは速さを 0 に近づけます。曲のキーが分からなければ「キー検出」を押すと入ります。 Match the key and scale to the song, then start at Amount 80% and Speed 30%. Move Speed toward 0 for the hard robotic effect. If you do not know the key, press Detect key and it fills itself in.

Mac では使えますかDoes it run on macOS?

CI で macOS 版もビルドしていますが、署名も動作確認もしていません。Windows 版を正式サポート対象としています。 CI produces macOS builds, but they are unsigned and untested. Windows is the supported platform.

アコグッジオはもう使えないのですかCan I still use AcoGzzio?

配信は終了しましたが、すでにインストール済みのものは動きます。今後の修正や新機能はボーカルグッジオにのみ入ります。 Distribution has stopped, but an installed copy keeps working. All future fixes and features go into VocalGzzio.
